Detailed Description

These are the levels of log verbosity used in a log call.

Member Data Documentation

Error

A log level for (severe) error messages.

It is suppressed only if the domains log level is set to DomainLevel.Off.

Info

The standard log level for normal log output statements.

Logged if the domains level is either DomainLevel.All or DomainLevel.InfoWarningsAndErrors.

Verbose

The most verbose log level to be used for debug output statements.

Logged only if the domains log level is set to DomainLevel.All.

Warning

A log level for warning messages, hence things that might lead to errors or are not welcome for other reasons, but maybe are not errors.

It is not logged only if the domains log level is DomainLevel.Errors orLogDomain.Level.Off.
