Class Description

The standard ReportWriter writing the message to Console.Error and Console.Out. A formatter of type FormatterPythonStyle is used to process the objects in the report message.

§ Formatter

Formatter Formatter = new FormatterPythonStyle()

The formatter used to format the output. This field is public and might be exchanged by users, while no 'safe' change interface is provided (neither in respect to threading, nor allocation. In contrast the field is just public. Therefore, when replacing this field,

  • the previous one needs to be deleted and
  • concurrent access needs to be avoided.
