Class Description

Specialization of abstract interface class ConfigurationPlugIn, which takes all command line parameters in the constructor and reads variable values from those parameters on request. Its priority value is fixed to 10.

Variable categories are used as a prefix together with an underscore '_'. This means, if variable IDE in category ALIB is accessed, the command line parameter –ALIB_IDE=xyz is read.

Command line variables may be passed with either one hyphen ('-') or two ('–'). Both are accepted.

Category and Variable names are insensitive in respect to character case.
