31 /** ************************************************************************************************
32  * This class provides an interface into system dependent timer values (usually 64 bit
33  * tick counters that are called \e "ticks" with \alib).
34  * Sibling class \alib{time,TimeSpan} and this class allow to store and calculate time
35  * values in an efficient and highly accurate way.
36  *
37  * As ticks are system dependent, they should not be stored and loaded in data files or
38  * otherwise shared between systems. Even on the same machine, the values may become
39  * incompatible (e.g. after a reboot).
40  * Also, in general such values should only be used by methods of
41  * this class and classes
42  * \alib{time,StopWatch},
43  * \alib{time,TimeSpan} and
44  * \alib{time,CalendarTime}.
45  *
46  * To construct an object of this type using calendar date an time values, helper class
47  * \b %CalendarTime with its constructor
48  * \alib{time,CalendarTime::CalendarTime(int,int,int,int,int,int,int)} is available. A typical
49  * construction with this helper could look like this:
50  *
51  * TimeStamp myDate= CalendarTime(2018,1,31,14,5).Get( Timezone::UTC );
52  *
53  *
54  * \note
55  * The resolution and accuracy of the values is platform dependent. Especially nanoseconds are
56  * deemed to be not accurate and above 100 ns (this was written and fact in 2013).
57  *
58  * \attention
59  * The dates storable in objects of this class is limited to a certain time range. In the
60  * current GNU/Linux and Windows implementations the range is roughly <c>+-292.27</c> years before
61  * and after the point in time that the system that the software is running on was initialized
62  * (booted). This value results from dividing <c>2‚Å∂63 bit</c> by the number of nanoseconds of
63  * one year of 365.25 days and the fact that internal, system dependent ticks counters are reset
64  * to \c 0 with the boot of a system.
65  **************************************************************************************************/
