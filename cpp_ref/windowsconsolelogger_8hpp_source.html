30 /** ************************************************************************************************
31  * A logger that logs all messages to the standard output <em>cout</em>.
32  * The name of the \e Logger defaults to "WINDOWS_CONSOLE".
33  *
34  * \b %ALox text logger escape sequences (see class \ref aworx::lox::ESC "ESC")
35  * are translated to Windows API calls (<em>SetConsoleTextAttribute</em>) which manipulates
36  * colors the text output.
37  * There is not 100% match between windows console capabilities and the definitions in
38  * \ref aworx::lox::ESC "ESC". Especially, ESC style attributes are ignored.
39  *
40  * Foreground and background colors can be set to be either light/dark or dark/light. This improves
41  * the readability of log output a lot and even allows to read if foreground and background colors
42  * are the same (they then still differ). However, the right setting for this is dependent on
43  * the color scheme of the final output device (window). To manipulate the right setting, see field
44  * #UseLightColors and also configuration variable
45  * [ALOX_CONSOLE_LIGHT_COLORS](../group__GrpALoxConfigVars.html).
46  *
47  * In the constructor, a default format string and some other definitions in member
48  * \ref MetaInfo get set to include ESC sequences. Of-course, these publicly accessible
49  * format attributes can be customized after creation.
50  *
51  * \note This class can not enable the output console (which receives \b %ALox
52  * log data) to support the windows API call <em>SetConsoleTextAttribute</em>.
53  * The opposite is right: this class should be used only if the output console supports such calls.
54  **************************************************************************************************/

79  /**
80  * Foreground and background colors chosen by this class might differ in their intensity.
81  * This increases the overall readability by increasing the contrast.
82  * If the background color of a console window is dark, then the background colors of
83  * colored log output should be darker colors than the foreground colors - and vice versa.
84  *
85  * Depending on the setting of this field, \b %ALox
86  * \ref aworx::lox::ESC "escape codes" for colors are translated to normal ANSI colors or
87  * lighter ones:
88  * - If this field is \ref LightColorUsage "LightColorUsage::Never", light colors are
89  * never used.
90  * - If this field is \ref LightColorUsage "LightColorUsage::ForegroundLight", foreground
91  * colors will be light colors and background colors dark ones. This is the default.
92  * - If \ref LightColorUsage "LightColorUsage::ForegroundDark", background colors will be
93  * light colors and foreground colors dark ones.
94  *
95  * The configuration variable [ALOX_CONSOLE_LIGHT_COLORS](../group__GrpALoxConfigVars.html)
96  * allows to externally modify this flag. It is read once within the constructor .
97  */
98  LightColorUsage UseLightColors;
