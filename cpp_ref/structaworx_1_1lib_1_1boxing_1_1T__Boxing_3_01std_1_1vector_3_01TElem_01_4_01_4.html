Class Description

template<typename TElem>
struct aworx::lib::boxing::T_Boxing< std::vector< TElem > >

[DOX_ALIB_BOXING_STD_VECTOR]

Specialization of struct T_Boxing for template type std::vector<T, std::allocator<T>> Instead of boxing a pointer to the vector object, a boxed array is stored, hence a pointer to the first element contents and the array length.

To suppress this default boxing behavior for type std::vector, use compilation symbol ALIB_FEAT_BOXING_STD_VECTOR_OFF.
