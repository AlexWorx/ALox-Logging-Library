aworx::lib::strings::boxing Namespace Reference

Namespace Description

This namespace provides a customization of ALib Boxing for ALib Strings. This means, on the one hand that code in this namespace specifies how objects found in parent namespace aworx::lib::strings are boxed and on the other hand introduces box-interfaces related to strings.

Attention
In this documentation, you will find specializations of templated namespace functions of namespace aworx::lib::boxing. This is of-course wrong in respect to the C++ language definition, which requires such specializations to be placed in the original namespace.
It was decided to 'trick' the documentation parser to show these specialization in this namespace, to keep namespace boxing clean and to have all specializations related to this namespace in one place.

void Init ( )

Initializes the ALib Boxing system in respect to boxing types found in module ALib Strings.
With the use of full ALib library, this method is invoked by ALIB::Init. If ALib Strings and ALib Boxing are used as separated modules, then the user has to invoke this somewhere in the bootstrap code.
