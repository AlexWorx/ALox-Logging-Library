Class Description

template<typename T>
struct aworx::lib::util::StringTreeConstKeyAllocator< T >

Implementation of StringTreeDefaultKeyAllocator which does not allocated keys. The original buffer is just returned.

This helper can be used in cases when child names and creation path strings used with StringTree and especially with the node creation methods of class StringTree::Cursor are 'safely' allocated already. This is for example true in the trivial case of using literal strings for child creation.
