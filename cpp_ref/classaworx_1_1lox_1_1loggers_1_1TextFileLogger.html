Class Description

This is a very simple file logger for textual log outputs. The file name string provided in the constructor is not verified. If write operations fail, this logger disables itself by setting the inherited flag isDisabled to true. The fileName may be changed by simply setting the public member FileName and the flag IsDisabled may be set to false by the user without the need of any other interaction other than acquiring the Lox that the logger is added to.
