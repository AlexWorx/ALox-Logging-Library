30 /** ************************************************************************************************
31  * This class provides an interface into system dependent timer values (usually 64 bit
32  * tick counters) that we call *ticks*. Ticks and this class allow to store and calculate time
33  * values in an efficient and highly accurate way.
34  *
35  * Ticks generated by this class can be added and subtracted:
36  *
37  * - Subtraction generates a *span value in ticks*
38  * - *Span value in ticks* can be added to themselves safely
39  * - *Span value in ticks* can be added to ticks safely
40  *
41  * Conversion functions to milliseconds, microseconds and nanoseconds accept ticks and span
42  * values in ticks in the same manner.
43  *
44  * As Ticks are system dependent, they should not be stored and loaded in data files or
45  * otherwise shared between systems. Even on the same machine, the values may become be
46  * incompatible (e.g. after a reboot).
47  * Also, in general such values should only be used by methods of
48  * this class and classes
49  * \ref aworx::lib::time::TickWatch "TickWatch",
50  * \ref aworx::lib::time::TickSpan "TickSpan" and
51  * \ref aworx::lib::time::TicksCalendarTime "TicksCalendarTime".
52  *
53  * The resolution and accuracy of the values is platform dependent. Especially nanoseconds are
54  * deemed to be not accurate and above 100 ns (this was written and fact in 2013).
55  **************************************************************************************************/
