Class Description

This class mimics class Thread of the Java runtime library. For general information of multi-threading support provided by ALib, see aworx::lib::threads.

bool IsAlive ( )
inline

Returns true, if this thread was started and is still running. If false is returned, the thread object can be safely deleted, without causing a blocking operation.

Note
For system threads (those not created using ALib thread features) true is returned. It can't be determined if the thread is still alive or not.

void Start ( )

Starts the execution of the thread. The thread's Run method is invoked, which (if not overwritten) invokes the Run method of any given Runnable. After returning from the method, unless the thread IsAlive will return true.
