48  /** ****************************************************************************************
49  * Logs an \b %aworx::Exception. If available, the correct source information is used for
50  * each entry.
51  *
52  * Parameter \p domainPrefix and \p logPrefix are both set on outer thread scope (using
53  * \ref aworx::lox::Lox::SetDomain "SetDomain( domainPrefix, Scope::ThreadOuter )" and
54  * \ref aworx::lox::Lox::SetPrefix "SetPrefix( logPrefix, Scope::ThreadOuter )".
55  * The reason for this approach is that it may occur that other prefixes or scope domains
56  * are set set on source scope for the source files and methods found in the exception
57  * entries' source code information. This is especially important to understand in respect
58  * to the log domain: With the default value of parameter \p domainPrefix being \e "/ERR",
59  * all source related domains will appear as a subdomain of \e "/ERR" when the exception
60  * is logged using this tool function.
61  *
62  * @param lox The lox to log into.
63  * @param e The exception to log.
64  * @param verbosity The verbosity of the log output.<br>
65  * Defaults to \b Verbosity::Error.
66  * @param domainPrefix The domain to log into. If \c NullString, no domain is set and
67  * the default domain is used instead.<br>
68  * Defaults to \e "/ERR".
