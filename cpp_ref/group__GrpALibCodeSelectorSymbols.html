ALib Symbols To Select Code Fragments

Detailed Description

Symbols (C++ preprocessor macros) listed here, are mostly generated out of ALib Compilation Symbols. In this case, their name corresponds to these symbols, excluding the postfixes _ON and _OFF.

These symbols must not be passed to the compiler and must also not be set manually from within code entities that use ALib libraries. They can be used in code entities that use ALib to select code depending on the same settings of ALib compilation. Selection takes place, using preprocessor directives:

#if defined(ALIB_XYZ_SELECTOR)
// your selective code
#endif
Note
Besides ALib-specific symbols of that type, ALib internally uses also platform specific symbols to select code, e.g. GNUC, _MSC_VER to detect compilers and GLIBCXX, _WIN32 to detect libraries used.

ALib Symbols are prefixed with ALIB_.

Macro Definition Documentation

#define ALIB_DEBUG

If defined, plausibility checks and ALib assertions are enabled. See ALIB_DEBUG_ON and ALIB_DEBUG_OFF for information about how to manipulate this symbol.

#define ALIB_DEBUG_STRINGS

Selects extra debug code within class AString. Gets defined by compiler symbol ALIB_DEBUG_STRINGS_ON.

#define ALIB_ECLIPSE_CDT

Used to select code fragments which are specific to running (and debugging) software from within the IDE Eclipse with CDT.

#define ALIB_FEAT_THREADS

Selects code within ALib (or source entities that use ALib) that cares about threads and due locking of shared resources. See compiler symbol ALIB_FEAT_THREADS_OFF for more information.

#define ALIB_GTEST

Selects unit test code in accordance to the Google Test libraries. Use ALIB_GTEST_ON to define this symbol.

#define ALIB_QTCREATOR

Used to select code fragments which are specific to running (and debugging) software from within the IDE QT Creator.

#define ALIB_VSTUDIO

Used to select code fragments which are specific to running (and debugging) software from within Microsoft Visual Studio.

For example, ALib unit tests are using this symbol to select the Visual Studio unit test framework. Use ALIB_VSTUDIO_ON to define this symbol.
