std_StringHashIgnoreCase Struct Reference

#include <string.hpp>

Class Description

Implements a case insensitive hash functor for class String, usable with types found in namespace std.
Instead of implementing std::hash inside namespace std, this struct can be provided as template parameter, e.g. to std::unordered_map, for which a templated type definition is provided for with aworx::UnorderedStringMapIgnoreCase.
