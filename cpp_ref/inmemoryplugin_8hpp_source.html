44 /** ************************************************************************************************
45  * Specialization of abstract interface class #ConfigurationPlugin, which holds
46  * configuration variables in memory. Instances of this class are used to create two plug-ins
47  * within standard configurations:
48  * - One with priority
49  * \alib{config,Priorities,Priorities::DefaultValues} (low priority)
50  * to store default values not found in other configuration plug-ins and
51  * - a second with priority
52  * \alib{config,Priorities,Priorities::ProtectedValues} (maximum priority),
53  * which can be used protect configuration values from external changes.
54  *
55  * This class in addition acts as the parent of class
56  * \ref aworx::lib::config::IniFile "IniFile" and potentially other (custom) classes.
57  * For this reason, it divides the set of variables into sections (according to the category),
58  * allows comment strings for variables and sections, and virtualizes some key methods to
59  * allow descendants to take specific actions.
60  *
61  * This class offers important internal fields and types for public access (bauhaus code style).
62  * However, in standard cases, only the interface methods of this class should be used.
63  **************************************************************************************************/
