45 /** ************************************************************************************************
46  * Specialization of abstract interface class #ConfigurationPlugin, which holds
47  * configuration variables in memory. Instances of this class are used to create two plug-ins
48  * within standard configurations:
49  * - One with priority
50  * \ref aworx::lib::config::Configuration::PrioDefault "PrioDefault" (low priority)
51  * to store default values not found in other configuration plug-ins and
52  * - a second with priority
53  * \ref aworx::lib::config::Configuration::PrioProtected "PrioProtected" (highest priority),
54  * which can be used protect configuration values from external change.
55  *
56  * This class in addition acts as the parent of class
57  * \ref aworx::lib::config::IniFile "IniFile" and potentially other (custom) classes.
58  * For this reason, it divides the set of variables into sections (according to the category),
59  * allows comment strings for variables and sections, and virtualizes some key methods to
60  * allow descendants to take specific actions.
61  *
62  * This class offers important internal fields and types for public access (bauhaus code style).
63  * However, in standard cases, only the interface methods of this class should be used.
64  **************************************************************************************************/
