Class Description

template<typename T, typename TKeyAllocator = StringTreeDefaultKeyAllocator<T>, char TSeparator = '/'>
class aworx::lib::util::StringTree< T, TKeyAllocator, TSeparator >::StdIterator

This inner class implements a std::iterator. This StdIterator is very simple compared to what class StringTree::Walker provides. Hence it is much more lightweight than class Walker.
Its primary use is for algorithms that just need to 'visit' all nodes of the StringTree (or a branch of it), regardless of the order. Hence, it is typically used in range-for-loops with std::for_each. This is why methods begin and end of class StringTree return a StdIterator of this class type.

For more information on iterators see description of outer class StringTree.
