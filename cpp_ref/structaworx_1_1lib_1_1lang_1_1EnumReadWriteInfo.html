Class Description

template<typename TEnum, typename TEnableIf = typename std::enable_if<T_EnumMetaDataDeclReadWrite<TEnum>::value>::type>
struct aworx::lib::lang::EnumReadWriteInfo< TEnum, TEnableIf >

When enums are equipped with enum meta data and in addition TMP struct T_EnumMetaDataDeclReadWrite is specialized, the enum elements become applicable to objects of class AString automatically through templated struct specializations T_Apply<TEnum> and T_Apply<TEnumBitwise>.
These methods write out the name of the enum element found within the static meta data table.

If in addition TMP struct T_EnumMetaDataPreAndPostFix is specialized for a type, the output is prefixed and postfixed with what methods Prefix and Postfix return.

See also
Macro ALIB_LANG_ENUM_PRE_AND_POSTFIX

◆ Prefix()

static String Prefix ( )
inlinestatic

The prefix string written before the element name. Used with methods T_Apply<TEnum> and T_Apply<TEnumBitwise> (or a custom write method).

If T_EnumMetaDataPreAndPostFix is not specialized for TEnum, an empty string is returned.
If T_Resourced is specialized for TEnum, then the value of T_EnumMetaDataPreAndPostFix::Prefix is appended to T_Resourced::Name and used as the resource name to load.

Otherwise, T_EnumMetaDataPreAndPostFix::Prefix is returned "directly".

Returns
The prefix string.
