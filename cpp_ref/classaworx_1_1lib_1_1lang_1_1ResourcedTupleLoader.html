ResourcedTupleLoader Class Reference

#include <resourcedtupleloader.hpp>

Class Description

Static helper class which loads vectors of arbitrary std::tuples (with supported element types) from Resources.

This class and static method LoadTable is used by implementations of EnumMetaData::CheckLoad to load enum meta data from resource strings.

Loads a vector of std::tuple objects from the resources.

The entries are either all read from resource variable named name, or, if this is not found, this methods searches for all resource variables named nameN with N starting at 0 and increasing until resource string nameN is not found.

The types of elements supported are:

  • aworx::String:
    Read 'as is' from the resource string.
  • char:
    Read 'as is' from the resource string.
  • Integer types:
    Read using singleton NumberFormat::Computational which supports decimal, binary, octal and hexadecimal formats.
Note
While all other types are simply ignored and default-constructed, still a separation character has to exist in the resource string.
Parameters
tuplesThe vector of tuples to fill
libraryThe library object to read from.
nameThe name of the resource string in resources.
delimThe delimiter of the values. Defaults to ','.
