C++ Reference

Welcome to ALox for C++!

This part of the ALox Logging library documentation is specific to its C++ Implementation.
ALox for C++ uses (and requires) C++ 11.

Attention
As of release version 1805 (May 2018), the C++ sources of ALox have been moved to an own, dedicated repository. The rationale for this was, that the relationship of ALib and ALox has been reversed: In the C++ world, ALox is now one of several modules of the general purpose library ALib, instead of building on it.

Documentation

With the transition of the C++ source code into a dedicated project and repository, the documentation of ALox is consequently located in two different places:

Documentation located at the ALox website (this):

  • Walk yourself through the ALox for C++ Tutorial. This is allows you a fast overview and intro if you are new to ALox.
  • Consult the general (language agnostic) ALox User Manual for in-depth information on various topics.
  • Information on configuring ALox for different C++ IDEs is provided with IDE Setup for ALox for C++.
  • ALox uses some essential preprocessor macros that form the 'highest' level API into ALox. Such information is provided with: ALox Macros

Documentation located at the ALib Class Library Website:

  • IDE/Project Setup of ALib for C++
  • Change Log
  • C++ Reference Documentation of ALox classes created with marvelous Doxygen!
  • Documentation of other ALib modules
