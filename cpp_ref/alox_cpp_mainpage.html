ALib

ALox builds on ALib which is a general utility library available in C++, C# and Java language variants. Next to the provision of basic system and language related functionality, the motivation for creating ALib (in contrast to using some existing 3rd party library) is to have a similar interface and a certain level of source code compatibility between the different language versions of ALox.

ALib is bundled with ALox and currently not released as a separate project. An exclamation to this bundled release are the so called ALib Modules. They are provided for C++ only and each provide a subset of ALib. Fore more information, refer to the ALib Documentation.

Organization of code

ALib classes are found in namespace

  • aworx::lib

and sub namespaces.

ALox itself consist of three namespaces. The main classes that the user of ALox interfaces with reside in:

  • aworx::lox

Some specific loggers are found in:

  • aworx::lox::loggers

Internal classes are located in

  • aworx::lox::core
