Class Description

template<typename TEnum>
struct aworx::lib::strings::applyto::T_Apply< TEnum >

Templated specialization of TMP struct T_Apply for enum types which meet the following conditions:

  • A specialization of TMP struct T_EnumMetaDataDeclReadWrite exists whose member NameIndex does not evalute to 0.
  • No specialization of TMP struct T_EnumIsBitwise exists that inherits from std::true_type.

(For bitwise enum types a separated specialization of T_Apply is provided.)
The conditions are evaluated by the compiler using std::enable_if on optional template parameter TEnableIf of T_Apply.

Member function Apply writes the name of the given enumeration value to target. To achieve this, the static vector of tuples provided with the specialization of TMP struct T_EnumMetaDataDecl is used as an "enum element to enum name" translation table.

As with all specializations of TMP struct T_Apply, the use of it is done implicit.

Searches the the value of src in the name table provided with field Table of a specialized version of that TMP struct and writes it to target.

Prior to the element name, if specialized for TEnum, a type prefix received with EnumReadWriteInfo::Prefix is written, as well as the corresponding postfix EnumReadWriteInfo::Postfix.

See class documentation for more information and a sample.
