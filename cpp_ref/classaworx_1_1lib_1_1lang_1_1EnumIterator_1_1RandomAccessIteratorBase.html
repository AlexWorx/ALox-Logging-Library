Class Description

template<typename TEnum, typename TEnableIf = typename std::enable_if<aworx::lib::lang::T_EnumIsIteratable<TEnum>::value>::type>
template<typename TPointer, typename TReference>
class aworx::lib::lang::EnumIterator< TEnum, TEnableIf >::RandomAccessIteratorBase< TPointer, TReference >

Implementation of std::iterator for enum type TEnum. This class exposes ConstIterator which uses const TEnum* and const TEnum& as pointer and reference types.

As the name of the class indicates, this iterator satisfies the std library concept RandomAccessIterator.
