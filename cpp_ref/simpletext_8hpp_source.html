43  * that starts at a certain index of an \b %AString buffer and reaches to its end.
44  *

46  * buffer and the parameters which are needed for the static method. With methods
47  * #Add, member #Formatter is used to append the given parameters and then format the

144  /** ****************************************************************************************
145  * Formats one or more paragraphs (separated by \b NewLine symbols) with three optional
146  * features:
147  *
148  * - Wrapping of lines longer than lineWidth (word wrap)
149  * - Justify the text, which here means "full justification", i.e. format the text to have
150  * lines of the exact same width.
151  * - Adding an indentation to each line with an optionally different indentation for
152  * the first line after a \b NewLine symbol and subsequent ones.
153  *
154  * The paragraph starts at \p startIdx and all of the rest of the string is treated
155  * as one paragraph. New-line character sequences found within the paragraph are considered
156  * manual line ends. Hence, no block formatting for lines ending with a new line character
157  * is performed.
158  *
159  * The method is static and hence can be used with arbitrary buffers.
160  * Non-static methods #Add invoke this methods after adding the given content
161  * to the internal buffer. Furthermore, convenience functions and corresponding member
162  * variables, simplify the use of this method, when indirectly used through an instance of
163  * the class.
164  *
165  * @param text The text containing the paragraph to format.
166  * @param startIdx The start of the paragraph.
167  * @param lineWidth The width of the line. If \c 0 or negative, no line wrap is
168  * performed.
169  * @param justifyChar If this is unequal to <c>'\0'</c> it denotes the fill
170  * character used to justify the paragraph. If <c>'\0'</c>,
171  * @param[out] maxLineWidth Provides the maximum width of all text lines written.
172  * paragraph is not justified.
173  * @param indentFirstLine The indent string of the first line. Defaults to \c nullptr.
174  * @param indentOtherLines The indent string of subsequent lines. Defaults to \c nullptr.
175  *******************************************************************************************/

209  /** ****************************************************************************************
210  * This method implements a pre-processing of the text prior to #Add paragraphs found in
211  * the text.
212  *
213  * The pre-processing is quite simple. Its purpose is to allow longer strings (e.g. loaded
214  * from resources) with multiple paragraphs to be formatted by embedded escape sequences
215  * to include indents and nested bullet schemes.
216  *
217  * The escape sequences begin with the character stored in field #MarkerChar, which defaults
218  * to <c>'@'</c>. The following table documents the sequences:
219  *
220  * Sequence | Description
221  * ---------|------------------------------------------------------------------------
222  * \@@ | Inserts the marker character itself.
223  * \@>> | Indent text by two spaces
224  * \@<< | Un-indent text by two spaces
225  * \@*> | Increases bullet level.
226  * \@<* | Decreases bullet level
227  * \@P | Inserts a new line (like '\n') but without ending the current and starting a new bullet point.
228  * \@HLc | Inserts a horizontal line of width #LineWidth using \p c as fill character.
229  *
230  * The nested bullet point characters are received from vector #MarkerBullets.
231  *
232  * @throws aworx::lib::strings::format::Exceptions
233  * - \alib{strings::format::Exceptions,UnknownMarker}
234  * - \alib{strings::format::Exceptions,EndmarkerWithoutStart}
235  *
236  * @param args The list of arguments to add.
237  ******************************************************************************************/
238  ALIB_API
239  void AddMarked( Boxes& args );
