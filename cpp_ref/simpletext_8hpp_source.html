34 /** ************************************************************************************************
35  * The class provides static method #FormatParagraph to format a "paragraph"
36  * that starts at a certain index of an \b %AString buffer and reaches to its end.
37  *
38  * When an instance of the class is created, the (public) members of the class provide the text
39  * buffer and the parameters which are needed for the static method. With methods
40  * #AddParagraph, member #Formatter is used to append the given parameters and then format the
41  * new paragraph. This way, a longer text might be built by repetitive calls.
42  **************************************************************************************************/

92  /** ****************************************************************************************
93  * Formats a paragraph by three optional things:
94  * - Adding an indentation to each line with an optionally different indentation for
95  * the first line
96  * - Wrapping of lines longer than lineWidth (word wrap)
97  * - Justify the text, which here means "full justification", i.e. format the text to have
98  * left and right aligned paragraphs.
99  *
100  * The paragraph starts at \p startIdx and all of the rest of the string is treated
101  * as one paragraph. New-line character sequences found within the paragraph are considered
102  * manual line ends. Hence, no block formatting for lines ending with a new line character
103  * is performed.
104  *
105  * The method is static and hence can be used with arbitrary buffers.
106  * Non-static methods #AddParagraph invoke this methods after adding the given content
107  * to the internal buffer.
108  *
109  * @param text The text containing the paragraph to format.
110  * @param startIdx The start of the paragraph.
111  * @param lineWidth The width of the line. If \c 0 or negative, no line wrap is
112  * performed.
113  * @param justifier If this is unequal to \c '\0' (the default), denotes the fill
114  * character used to justify the paragraph.
115  * @param indent The indent string.
116  * @param indentFirstLine The indent string of the first line. If this is \e nulled, than
117  * \p indent is used.
118  *******************************************************************************************/
119  ALIB_API
120  static
121  void FormatParagraph( AString& text, integer startIdx, integer lineWidth, char justifier,
122  String indent, const String& indentFirstLine );
