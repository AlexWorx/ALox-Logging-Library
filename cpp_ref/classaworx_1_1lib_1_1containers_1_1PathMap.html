Class Description

template<typename StoreT>
class aworx::lib::containers::PathMap< StoreT >

This class implements a map suitable for keys that are hierarchically organized. While the class works with any type of keys, it is efficient only if keys are rather long and similar to each other. Each portion of any key inserted is stored only once.

Note
This class is not considered as complete and final in the current version of ALib. The functionality currently is restricted to the needs of ALox logging library.
