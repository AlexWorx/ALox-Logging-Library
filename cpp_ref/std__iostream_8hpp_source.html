40  /** ****************************************************************************************
41  * Parameter class used for \e application to objects of type
42  * \ref aworx::lib::strings::AString "AString" with specialization of template struct
43  * \ref aworx::lib::strings::thirdparty::std::T_Apply<thirdparty::std::ReadLineFromIStream> "T_Apply<ReadLineFromIStream>".<br>
44  * Reads a line of text from a \b std::istream and appends it to the target \b %AString.<br>
45  * While, of-course, this class can be created 'inline' (similar objects of parameter classes
46  * found in \ref aworx::lib::strings::Format "Format"), if a series of lines are to be read
47  * from a \b std::istream, a local object of this type might be created outside the loop
48  * and used inside.

53  * - For a sample, refer to source code of ALib class \b %IniFile, method
