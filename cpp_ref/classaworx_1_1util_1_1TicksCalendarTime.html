Detailed Description

This class evaluates the calendar date and time from given Ticks object (representing a point in time). Consequently, it takes a Ticks object as a parameter. This class can be seen as an extension to the Ticks class which provides methods to get the total number of these values. In addition, the reverse calculation can be performed, which means, the field values can be set and the corresponding point in time as ticks received.
Finally, a method to format the date and time into string is available. This class is using system specific calendar methods and relies on the locale and time zone settings of the machine.

void Format ( const char *  format,
AString &  dest 
)

Formats the date using a given pattern string. Within the pattern string, different symbols are interpreted as tokens. The format is compatible with C# time format strings, as well as with class SimpleDateFormat of the Java APIs.
Strings within the format text that should not be interpreted as tokens may be surrounded by single quotes. Two single quotes in a row, will be replaced by one single quote.
The following tokens are supported:

y The year with as many digits as it has (for current dates this is 4).
yy The year, truncated to 2 digits (modulo 100).
yyy...y The year with a minimum amount of digits as amount of y-characters given.
M The month as numbers from 1..12.
MM The month as numbers from 01..12.
MMM The month as abbreviated, 3-digit word in English language.
MMMM The month as word in English language.
d The day as numbers from 1..31.
dd The day as numbers from 01..31.
ddd The day as abbreviated, 3-digit word in English language.
dddd The day as word in English language.
H The hour as numbers from 0..23.
HH The hour as numbers from 01..23.
K The hour as numbers from 0..11 am/pm.
KK The hour as numbers from 01..11 am/pm.
m The minute as numbers from 0..59.
mm The minute as numbers from 01..59.
s The second as numbers from 0..59.
ss The second as numbers from 00..59.
