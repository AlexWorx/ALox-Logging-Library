Class Description

Parameter class which is 'applicable' to objects of type AString with corresponding specialization of template function ApplyTo.

Reads a line of text from a std::istream and appends it to the target AString.
While, of-course, this class can be created 'inline' like most ApplyTo parameter classes, if a series of lines are to be read from a an std::istream, a local object of this type might be created before the read loop and used inside. In this case, the 'output parameter' IsEOF, can be used e.g. as a loop criteria.

See also
  • operator>>(std::istream&, AString&) and operator<<(ostream& os, const String& as)
  • For a sample, refer to source code of ALib class IniFile, method IniFile::ReadFile.
