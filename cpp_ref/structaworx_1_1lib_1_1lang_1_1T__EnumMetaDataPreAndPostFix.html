Class Description

template<typename TEnum, typename TEnableIf = typename std::enable_if<T_EnumMetaDataDecl<TEnum>::value>::type>
struct aworx::lib::lang::T_EnumMetaDataPreAndPostFix< TEnum, TEnableIf >

This TMP struct is used in combination with template class EnumReadWriteInfo and its methods Prefix and Postfix. These methods are invoked for example by T_Apply<TEnum> and T_Apply<TEnumBitwise> (or custom code) to retrieve and write a prefix and postfix string around a writable enum element.

Attention
In the case that T_Resourced is specialized for type TEnum, methods EnumReadWriteInfo::Prefix and EnumReadWriteInfo::Postfix use the fields Prefix and Postfix of this struct as a postfix that is appended to the resource name for the strings and loads the values from the resources.

Otherwise, the fields are directly returned.

Macro ALIB_LANG_ENUM_PRE_AND_POSTFIX is used to specializes this struct.

Template Parameters
TEnumThe enumeration type.
TEnableIfTo be ignored. Used to select this struct only for enum types which have a specialization of T_EnumMetaDataDecl in place.
