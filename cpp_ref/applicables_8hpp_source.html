388  /** ********************************************************************************************
389  * This is a type applicable to objects of type
390  * \ref aworx::lib::strings::AString "AString". Various constructors consume integer and
391  * floating point types, along with formatting options. The specialization of struct
392  * \ref T_Apply "T_Apply<Format>"
393  * will use a given (or defaulted) instance of class
394  * \ref aworx::lib::strings::NumberFormat "NumberFormat" to append the value to the
395  * \b %AString in question.
396  *
397  * \note
398  * Within the same header file that this class in declared in, there are several
399  * specializations of \ref T_Apply defined for plain integer and
400  * floating point types. These specializations create an object of this type, providing the
401  * value only, hence, using this classes constructors' default values. The number format
402  * used as default by the constructors of this class is
403  * \ref aworx::lib::strings::NumberFormat::Computational "NumberFormat::Computational".
404  * As a result, the application of such core types, as in:
405  * \snippet "DOX_ALIB_APPLYTO.cpp" DOX_ALIB_APPLYTO_FORMAT1
406  * which produces:
407  * \snippet "DOX_ALIB_APPLYTO_FORMAT1.txt" OUTPUT
408  *
409  * \note
410  * does \b not use a locale specific number format. Instead it uses one that is exchangeable
411  * between applications independent from the locale setting of the executing machine.<br>
412  * Consequently, for locale specific output, an object of this class needs to be applied
413  * along with a locale enabled instance of \b %NumberFormat. For example:
414  * \snippet "DOX_ALIB_APPLYTO.cpp" DOX_ALIB_APPLYTO_FORMAT2
415  * which - dependent on the current local setting - might produce:
416  * \snippet "DOX_ALIB_APPLYTO_FORMAT2.txt" OUTPUT
417  *
418  *
419  * <b>Inner Types:</b><br>
420  * Besides accepting plain number types, this class in addition aggregates several public inner
421  * types, namely
422  * - \ref aworx::lib::strings::Format::Tab "Format::Tab",
423  * - \ref aworx::lib::strings::Format::Field "Format::Field",
424  * - \ref aworx::lib::strings::Format::Bin "Format::Bin",
425  * - \ref aworx::lib::strings::Format::Hex "Format::Hex" and
426  * - \ref aworx::lib::strings::Format::Oct "Format::Oct".

429  * \ref T_Apply and provide certain formatting options.

433  *
434  * <b>Details on Formats:</b><br>
435  * Details on the options of formatting integer and floating point numbers are documented
436  * with class
437  * \ref aworx::lib::strings::NumberFormat "NumberFormat".
