TickSpan Class Reference

#include <tickspan.hpp>

Class Description

This class represents a time span, measured in days, hours, minutes and so on. It provides methods to import from and to an object of type Ticks.
(Note: Objects of type Ticks may represent a point in time or a time span, dependent on what value they have set to).
