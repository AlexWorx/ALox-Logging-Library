Class Description

template<typename TPointer, typename TReference>
class aworx::lib::strings::String::RandomAccessIteratorBase< TPointer, TReference >

Implementation of std::iterator for class String. This class exposes ConstIterator which uses const char* and const char& as pointer and reference types. Descendent classes may expose a mutable version (e.g. AString).

As the name of the class indicates, this iterator satisfies the std library concept RandomAccessIterator.
