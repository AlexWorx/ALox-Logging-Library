Class Description

This is central class of the ALox logging implementation. It is not recommended to use this class directly for logging. Instead, use the simple and convenient static interface class Log or, for release logging and other more complex operations use a Lox instance. The class is abstract. To implement an own log stream, derive a new Logger class and implement the abstract method doLog().

This function is invoked by method Line(), only if this instance is not disabled and domain level and given level match. Sets the domains' log level and (by default) all it's sub domains recursively. In the case that sub domains should be set to a different log level, then this function has to be called for such sub domains after the call to the parent domain (or recursion has to be switched off, using the parameter propagation). It is not necessary to register/create a domain before setting its log level and log levels can be set and modified any time.

Note
It is recommended to set domain levels for one or more loggers using Lox::SetDomain, which invokes this method internally. This method may be used directly in cases when certain dedicated domains (e.g. Lox::InternalDomain are to be manipulated for a logger, regardless in which context (Lox) it is used.
Parameters
domainThe log domain.
domainLevelThe domains log level to be set.
propagation(Optional) If Propagation::ToDescendants, which is the default, the level is set for all sub-domains recursively. If Propagation::None, then only this domain is changed.

unsigned int CntLogs =0

The number of logs actually performed so far. In a text logger this is similar to the line number, despite the fact that a single log call can produce more than one line.

String32 Name

The name of the logger. Can be used to receive a reference to a logger and for filtering. Logger names are case insensitive. If no name is specified with the constructor, the name will by the same as TypeName.

Domain RootDomain

The root domain "/". All registered or just used domains become a sub domain of this root. If a sub domains log level is not explicitly set, such sub domain inherits the level setting of the root domain. Therefore, the log level setting of the root domain determines how unknown domains got logged. The default level of the root domain is Log.DomainLevel.Off.

String32 TypeName

The type name of the logger. This is equal to the default Logger name.
