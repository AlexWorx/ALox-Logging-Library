Detailed Description

Symbols (C++ preprocessor macros) listed here, are mostly generated out of AWXU Compilation Symbols. In this case, their name corresponds to these symbols, excluding the postfixes _ON and _OFF.

These symbols must not be passed to the compiler and must also not be set manually from within code entities that use AWXU libraries. They can be used in code entities that use AWXU to select code depending on the same settings of the AWXU library compilation. Selection takes place, using preprocessor directives:

#if defined(AWXU_XYZ_SELECTOR)
// your selective code
#endif
Note
Besides AWXU specific symbols of that type, AWXU internally uses also platform specific symbols to select code, e.g. GLIBCXX, _MSC_VER, etc.

AWXU Symbols are prefixed with AWXU_.

In addition to the AWXU specific symbols, the AWXU package and headers files also support symbols that are not specific to AWXU functionality but are really general symbols used in many projects. Those symbols are prefixed AWORX_.

Macro Definition Documentation

#define AWORX_GTEST

Selects unit test code in accordance to the Google Test libraries. Use AWORX_GTEST_ON to define this symbol.

#define AWORX_QTCREATOR

Used to select code fragments which are specific to running (and debugging) software from within the IDE QT Creator.

#define AWORX_VSTUDIO

Used to select code fragments which are specific to running (and debugging) software from within Microsoft Visual Studio.

For example, A-Worx software uses this symbol to select the Visual Studio unit test framework. Use AWORX_VSTUDIO_ON to define this symbol.

#define AWXU_DEBUG

If defined, plausibility checks and AWXU assertions are enabled. See AWXU_DEBUG_ON and AWXU_DEBUG_OFF for information about how to manipulate this symbol.

#define AWXU_DEBUG_ASTRING

Selects extra debug code within class AString. Gets defined by compiler symbol AWXU_DEBUG_ASTRING_ON.

#define AWXU_FEAT_THREADS

Selects code within AWXU (or source entities that use AWXU) that cares about threads and due locking of shared resources. See compiler symbol AWXU_FEAT_THREADS_OFF for more information.
