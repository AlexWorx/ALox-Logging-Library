Class Description

This class is only available in debug compilations of ALib and furthermore only if ALib Boxing is used in an ALib module distribution that incorporates module ALib Strings.

It uses the plain debug methods of classes Box and Boxer that accept and return type information structs std::typeinfo and converts them to strings and sorted lists of strings.
