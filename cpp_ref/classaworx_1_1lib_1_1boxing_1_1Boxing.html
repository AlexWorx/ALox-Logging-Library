◆ init()

void init ( Phases  phase)
overrideprotectedvirtual

Initializes the ALib Boxing system.

The method performs the following initialization steps:

  • Adds default interface class IEquals
  • Adds default interface class IIsNull
  • Adds default interface class IIsEmpty

If code selection symbol ALIB_FEAT_BOXING_FTYPES is 1, then in addition,

  • implementations of interface IEquals for fundamental types double, char, wchar_t, char16_t and char32_t and
  • implementations of interface IIsNull for fundamental types boxed_int, boxed_uint and double,

are registered.

Parameters
phaseThe initialization phase to perform.

Implements Library.
