151  /**
152  * The line format specifies the (automated) log output that is prepended to each log line before
153  * the log message itself. This format string can be changed after the creation
154  * of a \ref TextLogger.
155  *
156  * The string supports replacement variables that begin with a % sign
157  * - %SP: The full path of the source file
158  * - %Sp: The trimmed path of the source file
159  * - %SF: The callers' source file name
160  * - %Sf: The callers' source file name without extension
161  * - %SL: The line number in the source file
162  * - %SM: The method name
163  *
164  * - %TD: The date the log call was invoked
165  * - %TT: Time of day the log call was invoked
166  * - %TC: Time elapsed since the Logger was created or its timer was reset
167  * - %TL: Time elapsed since the last log call
168  *
169  * - %tN: Thread name
170  * - %tI: Thread ID
171  * - %V: The verbosity. This is replaced by the corresponding strings found in fields
172  * #VerbosityError, #VerbosityWarning, #VerbosityInfo and #VerbosityVerbose.
