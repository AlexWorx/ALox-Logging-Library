TimeSpan Class Reference

#include <timespan.hpp>

Class Description

This class provides an interface into system dependent timer values (usually 64 bit tick counters) that are called "ticks" with ALib. Sibling class TimeStamp and this class allow to store and calculate time values in an efficient and highly accurate way.

Often, objects of this class are generated by the subtraction of TimeStamp values or by using methods TimeStamp::Age and TimeStamp::Since.

As ticks are system dependent, they should not be stored and loaded in data files or otherwise shared between systems. Even on the same machine, the values may become incompatible (e.g. after a reboot). Also, in general such values should only be used by methods of this class and classes StopWatch, TimeStamp and CalendarTime.

The resolution and accuracy of the values is platform dependent. Especially nanoseconds are deemed to be not accurate and above 100 ns (this was written and fact in 2013).
