Class Description

template<size_t TLength>
struct aworx::lib::strings::boxing::T_Boxing< strings::StringLiteral< TLength > >

Templated specialization of template struct T_Boxing for class StringLiteral with different template TLength.
The types are boxed as character arrays, as any string-type should be.
