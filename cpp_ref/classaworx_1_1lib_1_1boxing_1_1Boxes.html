Class Description

A vector of objects of class Box. Specializes class std::vector<Box> (publically) with a constructor and methods to add a variable amount of arbitrary values with one invocation.

Chapter 7. Variadic Arguments of the namespace documentation.

void Add ( TBoxable &&  src)
inline

Template method that boxes and adds one value to this vector of boxes.

If the given value is an instance of this class, then, instead of "boxing" the list, the values of that list are each added. The latter also works if the given value is a boxed instance of this class.
