Class Description

Parameter class used for application to objects of type AString with corresponding specialization of template function ApplyTo.

Appends the contents of a string to to the AString using a defined 'field'-width. If given string value is shorter than parameter width specifies, the field is filled with a corresponding amount of padChar characters.
Parameter alignment of type Alignment allows to left-, right- or center-align the contents of the field.
