Class Description

template<typename T>
struct aworx::lib::util::StringTreeDefaultKeyAllocator< T >

This struct is used as the default implementation of a helper type of class StringTree. Class StringTree uses this class to allocate memory to store character sequences for child node names when child nodes are created.

This default implementation simply allocates the memory using new and copies the data. The interface of methods allocate and free provide the value of the node. There might be implementations where the value has already a copy of the child name allocated (or wishes to do). In this case, that string can be returned.

See also StringTreeConstKeyAllocator.
