Class Description

This is one of the built-in Interface classes of ALib Boxing. Similar to the corresponding virtual method of class Object in languages like Java and C#, this interface is invokable with all boxable types, as a default implementation is registered (see Boxer::DefineDefaultInterface).

The default implementation is implemented with this base interface class itself. For non-array types, this default implementation checks if both boxes share the same type and data, without interpreting the data in any way.
For array types, the contents of the array is checked using memcmp() if types and lengths are the same.
