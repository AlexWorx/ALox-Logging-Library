Class Description

template<typename Type>
struct aworx::lib::strings::ToStringDefined< Type >

This is a TMP (template meta programming) 'class' which defaults the question ToStringDefined<T>::value to false for every type.
When implementing specializations of template function ToString for custom types, this class has to be specialized as well. Otherwise, a compile time assertion (static_assert) will fail whenever the type is used to implicitly construct an String providing that type.

Definition has to be made as follows: 

    template<>  struct ToStringDefined<T> : public std::true_type {};

where T is the type that is to be defined given as a const reference. For example, in the ALib support header file "alib/compatibility/std_string.hpp", for class std::string, the definition looks as follows: 

    template<>  struct ToStringDefined<const std::string&> : public std::true_type {};

For more information and a sample refer to ToString.
