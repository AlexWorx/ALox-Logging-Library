Format::UInt32 Class Reference

#include <applyformat.hpp>

Class Description

Parameter class used for application to objects of type AString with corresponding specialization of template function ApplyTo.

Appends an uint32_t value using a minimum number of digits. If given constructor parameter value has less digits than parameter minDigits specifies, then a corresponding amount of '0' characters are prepended.
The given value minDigits is cut to the range 1..20 ( which equals the maximum digits of a 64 bit unsigned integer value, which is the largest integer processed in overloaded methods).
